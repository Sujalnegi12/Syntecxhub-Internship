# 📊 Sales Performance Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive **Sales Performance Dashboard** built using **Microsoft Power BI**. The dashboard analyzes sales data to provide meaningful business insights through visualizations, KPIs, and forecasting.

## 🎯 Objectives
- Import and clean the raw sales dataset
- Remove null values and duplicates
- Analyze monthly, quarterly, and yearly sales trends
- Identify top-selling and low-performing products
- Compare sales across different regions and categories
- Create KPIs such as Total Sales, Total Profit, Total Orders, and Growth Rate
- Generate sales forecasts for the next 15 days
- Provide actionable business insights through an interactive dashboard

## 🛠️ Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX
- Excel / CSV Dataset

## 📈 Dashboard Features
- Interactive filters and slicers
- Sales Trend Analysis
- Profit Analysis
- Category-wise Sales
- Region-wise Performance
- Top & Bottom Performing Products
- Monthly & Yearly Comparison
- Sales Forecast (15 Days)
- KPI Cards
- Actionable Business Insights
